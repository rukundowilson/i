Explore the software I rely on, the gadgets I'm passionate about, and other recommendations that I stand by.

Many people frequently ask about the tools and gadgets I rely on for software development, productivity, and daily life. I'm excited to share a detailed list of my go-to tools and tech that fuel my work and enhance my efficiency.

production

linux

To minimize external virus risks, I usually use Linux as my primary operating system. Linux's robust security features and its open-source nature contribute significantly to a safer computing environment.

Additionally, Linux offers a wide range of tools specifically designed for security testing and system monitoring. This versatility and depth of security options make Linux my preferred environment for both development and everyday use.

development

visual studio code

I usually use Visual Studio Code for writing all of my code. Its extensions are easy to configure, making my work more organized and efficient.

sublime text

Sublime Text is another excellent choice for writing code. I particularly appreciate its code formatting capabilities and its clean, appealing interface.

Artificial intelligence

Incorporating artificial intelligence into my projects is a valuable practice for me. I utilize AI to enhance efficiency and achieve better results, aligning with the growing trend of AI integration in various industries.

Designing

Figma

We started using Figma as just a design tool but now its become our virtual whiteboard for the entire company. Never would have expected the collaboration features to be the real hook.

motivation

headphones

I usually use headphones to minimize external distractions, stay motivated during development, and maintain focus.
